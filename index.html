FAQ

How is this different from Jasper / Taplio / Castmagic?

Those tools handle one step each. Jasper writes (badly — “below-average paragraphs,” per leapshq.com). Taplio schedules but doesn’t research. Castmagic repurposes but doesn’t originate. The Ambient Content Engine handles the full loop: signal scan → persona research → branded content → scheduled post. The integration is the product.

I don’t trust AI agents with access to my machine. Why should I trust this?

You shouldn’t trust blindly — that’s the point. ThoughtWorks says “do not run these agents on corporate endpoints.” The pipeline runs inside Bilrost, a purpose-built sandbox with 12 Ansible roles, 273 CLI tests, and a published STRIDE threat model. Security researcher Paul McCarty found malware in ClawHub skills within two minutes (aikido.dev). Our response: containerized execution with zero host access. Read the threat model yourself.

LinkedIn carousels feel gimmicky. Do they actually perform?

6.60% average engagement rate — 278% more than video, 596% more than text-only. These aren’t opinion numbers. LinkedIn’s own algorithm rewards carousel dwell time. The pipeline generates carousels because the data says to, not because they’re trendy.

I tried AI content tools and they all produce slop. Why would this be different?

Because slop comes from generating without research. The pipeline doesn’t start with “write me a LinkedIn post.” It starts with a signal scan of real conversations — Reddit threads, HN discussions, industry blogs — and extracts actual pain language from real people. Every post is grounded in evidence. The output sounds like a practitioner who did the research, because the research actually happened.

How do I know the skills won’t just stop working like every other agent integration?

Skills auto-activate ~20% of the time because they hit context limits silently (blog.fsck.com). The pipeline’s skills are designed for chained execution with explicit handoffs — each skill reads the previous skill’s output file, not context window memory. No silent failures. If a skill fails, the pipeline stops and tells you why.

I’m not technical. Can I still use this?

Tier 1 (articles) requires zero technical skill — just read. Tier 3 (managed pilot) requires zero technical skill — we run everything. Tier 2 (clone + run) requires CLI comfort and Claude Code. Pick your tier.

What if my content sounds generic even with persona data?

The pipeline includes a voice profile system (LinWheel). You create a voice profile from your existing writing — tone, vocabulary, sentence structure. Every generated post runs through your voice profile before publishing. The output sounds like you because it’s calibrated to you.
